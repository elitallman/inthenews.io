Welcome at PyTrending!

“... find what you were not yet looking for... ”

My goal here is to find whatever is most interesting in Python; today. It uses Reddit, Stackoverflow, GitHub, PyPi and Twitter currently. You can suggest new sources (and/or twitter streams) by emailing kootenpv@gmail.com.
I'll consider the possibility for people to sponsor certain python packages. I will never show advertisements, but a way to promote open source projects sounds awesome to me!
It could be forked for other languages, but Python deserves this first ;)

How it works

GitHub -- GitHub's trending page contains the packages that have the most stars that day. This is boring, because it's usually the same packages that are trending. I'm now preventing this, only showing new pages (with the occasional slip where a well established one is still mentioned).

Reddit -- Currently I have the terrible model that whenever something reaches 30 votes, it becomes trending

Stackoverflow -- Bounty questions with the python tag; sorted on date

pypi (pip) -- Here you can simply see the packages that are most recent (so actually not trending yet, will need historic data to predict trending)

Twitter -- just the latest tweets from one twitter channel at the moment. Feel free to suggest useful channels.
