How it works

GitHub -- just posts what is trending daily. This is boring, because it's usually the same packages that are trending. I'm now preventing this, only showing new (with the occasional slip).

Reddit -- Currently I have the pathethic model that whenever something reached 30 votes, it becomes trending

Stackoverflow -- Bounty questions with the python tag are sorted on date

pypi (pip) -- Here you can simply see the packages that are most recent (so actually not trending yet, will need historic data to predict trending)

Twitter -- just the latest tweets
